# Simple Todo List

A beginner-friendly Todo List project built using HTML, CSS, and JavaScript.

## Features

* Add new tasks
* Mark tasks as completed using checkboxes
* Completed tasks show with strikethrough effect
* Simple and clean UI

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

```
To-do list/
│
├── index.html
├── style.css
└── script.js
```

## How It Works

1. Enter a task in the input field
2. Click the **Add** button
3. Task appears in the list
4. Tick the checkbox to mark the task as completed

## Concepts Practiced

* DOM Manipulation
* createElement()
* Event Handling
* User Input Handling
* Basic JavaScript Logic
